+ Kurs __21 Şubat 2023 Salı__ günü başlayacaktır.

+ Kurs gerçek zamanlıdır.
+ Toplam kurs süresi __180__ saattir. Dersler __Salı ve Perşembe__ günleri __19:30 - 22:30__ saatleri arasında yapılacaktır.
